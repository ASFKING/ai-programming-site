# AI编程：智能开发的未来

> 从代码补全到智能体编程，AI正在重塑软件开发的方式。

一个系统化的 AI 编程知识站，帮助开发者全面了解 AI 编程技术、工具和方法论。

🌐 **在线访问**：[https://asfking.github.io/ai-programming-site/](https://asfking.github.io/ai-programming-site/)

---

## ✨ 特色内容

- **概念入门** — AI 编程发展历程、核心概念与技术演进
- **工具选型** — 覆盖 10 款主流工具的能力光谱对比与智能选型器
- **编程范式** — 四大范式解析（Copilot / Chat / Agent / 自主编程）
- **方法论** — 从 Chat 模式到 Spec 驱动开发的完整实践指南

## 🚀 本地开发

```bash
# 克隆仓库
git clone https://github.com/ASFKING/ai-programming-site.git
cd ai-programming-site

# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建静态站点
npm generate

# 预览构建结果
npm run preview
```

## 📁 项目结构

```
├── content/            # Markdown 内容文件
│   ├── concepts/       # 概念入门
│   ├── tools/          # 工具评测与对比
│   ├── methodology/    # 方法论
│   ├── paradigms/      # 编程范式
│   └── about/          # 关于本站
├── components/         # Vue 组件
├── composables/        # Vue 组合式函数
├── layouts/            # 页面布局
├── pages/              # 页面路由
├── plugins/            # Nuxt 插件
├── assets/             # 静态资源（样式、图片等）
├── public/             # 公共静态文件
└── nuxt.config.ts      # Nuxt 配置
```
