# OpenAI Codex

OpenAI Codex 是驱动 GitHub Copilot 的底层模型，OpenAI 也推出了其作为独立智能体的版本。

## 核心特性

- **端到端交付**：不仅生成代码，还擅长理解复杂的业务逻辑并分解为多个步骤执行。
- **强大的推理能力**：基于 GPT 系列大型语言模型，拥有顶级的逻辑推理能力。
- **多模型协作**：支持在不同的子任务中调度专门优化的模型。

## 适用场景

适用于需要**深度逻辑推理**和**半自主任务执行**的场景。虽然目前在 IDE 集成体验上不及其他工具，但在处理纯粹的编程逻辑问题时，它的表现依然非常强劲。
