# OpenAI Codex

Codex 是 OpenAI 推出的云端 AI 编程智能体，由 codex-1 驱动——一个针对软件工程优化的 o3 版本。

## 核心特性

- **codex-1 模型**：基于 o3 通过强化学习在真实编程任务上训练，代码风格贴近人类 PR 偏好。
- **多任务并行**：每个任务在独立的云端沙箱中运行，可同时处理多个任务。
- **端到端交付**：自主规划、编码、测试、调试，直到交付可用的代码。

## 适用场景

适用于需要**深度逻辑推理**和**自主任务执行**的场景。特别适合批量处理独立的编程任务，如功能开发、Bug 修复、代码审查等。
