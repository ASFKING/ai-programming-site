# CodeBuddy

CodeBuddy 是腾讯云推出的 AI 全流程智能编程工具，致力于构建产品、设计、研发、部署无缝协作的共生环境。支持 IDE、插件和 CLI 三种形态，覆盖从专业开发者到零基础用户的全场景需求。

## 产品形态

| 形态 | 适用用户 | 核心特点 |
|------|----------|----------|
| **CodeBuddy IDE** | 产品 / 设计师 / 全栈开发 / 编程初学者 | 产设研一体工作台，主打「对话即编程」 |
| **CodeBuddy 插件** | 日常编码开发者 / 特定 IDE 使用者 | 即插即用，融入现有工作流（VS Code / JetBrains） |
| **CodeBuddy Code（CLI）** | DevOps / 运维 / SRE / 资深开发者 | 命令行工具，支持 Shell/文件/网络操作，任务编排 |

## 核心特性

- **全流程 AI 驱动**：从智能需求分析（自然语言转 PRD）→ 设计稿生成 → 设计稿转代码 → 智能编程 → 一键部署，打通产品到上线的完整链路。
- **Figma 设计稿转代码**：内置 Figma 功能，直接将设计稿转化为可维护的前后端代码，打通设计到研发的「最后一公里」。
- **多模型支持**：支持混元、DeepSeek 等多种对话大模型。
- **BaaS 服务集成**：内置 Supabase 和腾讯 CloudBase，自动处理数据库、用户认证等后端服务。
- **一键部署**：支持部署至 CloudStudio、EdgeOne Pages 等平台。
- **MCP 协议支持**：可方便地扩展外部工具和知识库。

## 适用场景

- **产设研一体化开发**：产品、设计、开发角色在同一平台协同工作
- **设计稿转代码**：从 Figma 设计稿或手绘草图直接生成前端代码
- **腾讯云生态项目**：深度集成腾讯云服务，从编码到运维一站式体验
- **零基础用户**：通过「对话即编程」模式，无需编程基础即可构建应用
- **DevOps 自动化**：通过 CLI 形态在命令行中用自然语言驱动开发全流程
