## 核心功能

### Composer 多文件协同

- 跨文件编辑，AI 理解项目上下文
- 支持同时修改多个文件
- 自动处理文件间的依赖关系

### 8 个智能体并行

- 多个 AI Agent 同时工作
- 分工处理不同模块
- 大幅提升大型项目的开发效率

### 自定义模型端点

- 支持接入自定义 LLM
- 可配置 API 代理
- 灵活的模型切换
