# Windsurf 🏄

> Codeium 出品的 AI 原生 IDE，集成 Devin 云端智能体

## 核心功能

### Devin 云端智能体

- 将复杂任务委托给 Devin，它在独立机器上自主工作
- 你可以继续编码，Devin 同时处理其他任务
- 完成后自动提交 PR 供你审查

### Flow 模式

- 实时 pair programming 体验
- AI 像结对编程伙伴一样与你协作
- 自动理解项目上下文并提供建议

### Agent Command Center

- 统一管理本地 Cascade 和云端 Devin 会话
- Kanban 风格仪表板，一目了然
- Spaces 功能：围绕单个任务捆绑会话、PR、文件和共享上下文

### 性价比优势

- Pro 版 $15/月，比 Cursor 便宜 25%
- 功能覆盖度接近 Cursor
- 适合预算敏感的开发者

## 适用场景

- ✅ 需要委托复杂任务给云端智能体
- ✅ 实时 pair programming
- ✅ 全栈开发
- ✅ 预算敏感的开发者

## 局限性

- 需要科学上网
- 社区规模不如 Cursor
- 部分功能仍在迭代中
