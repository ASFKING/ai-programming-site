# Windsurf 🏄

> Codeium 出品的 AI 原生 IDE，以 Flow 模式著称

## 基本信息

| 项目 | 详情 |
|------|------|
| 开发商 | Codeium |
| 类型 | AI 原生 IDE |
| 定价 | 基础版免费，Pro $15/月 |
| 网络要求 | 需要科学上网 |
| 官网 | [windsurf.com](https://windsurf.com) |

---

## 核心功能

### Flow 模式

- 实时 pair programming 体验
- AI 像结对编程伙伴一样与你协作
- 自动理解项目上下文并提供建议

### Cascade 多步推理

- 能处理多步骤的复杂任务
- 自动分解任务并逐步执行
- 支持跨文件的上下文感知

### 性价比优势

- Pro 版 $15/月，比 Cursor 便宜 25%
- 功能覆盖度接近 Cursor
- 适合预算敏感的开发者

---

## 适用场景

- ✅ 实时 pair programming
- ✅ 全栈开发
- ✅ 需要流畅 AI 协作体验的开发者
- ✅ 预算敏感的开发者

## 局限性

- 需要科学上网
- 社区规模不如 Cursor
- 部分功能仍在迭代中
- 插件生态不如 VS Code 原版
