# Qoder

Qoder 是字节跳动推出的一款轻量级 AI 编程插件，与 Trae 互为补充。

## 核心特性

- **极致速度**：针对单行代码补全进行了深度优化，延迟极低。
- **资源占用低**：作为轻量级插件运作，不改变开发者的 IDE 使用习惯。
- **深度定制模型**：搭载字节跳动自研的代码大模型，在处理中文注释和国产技术栈时表现优异。

## 适用场景

适合那些不想更换 IDE（如已习惯 VS Code 或 JetBrains 原生体验），但又需要**快速、流畅的国内 AI 补全**服务的开发者。
