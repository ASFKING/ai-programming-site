# Qoder

Qoder 是阿里巴巴于 2025 年 8 月正式发布的新一代 Agentic 编程平台，基于 VSCode 开源代码深度定制并内置 AI 能力。核心理念是「定义目标，审查结果」。

## 四种对话模式

| 模式 | 定位 | 适用场景 |
|------|------|----------|
| **Ask** | 只读问答 | 学习、探索代码库、获取建议 |
| **Agent** | 自主编码 | 日常开发、多文件编辑、终端操作 |
| **Experts** | 多智能体协作 | 全栈开发、复杂调试、技术调研 |
| **Quest** | 全自主编程 | 端到端任务交付，最少人工干预 |

## Quest 全自主编程（核心亮点）

Quest 模式支持三种场景：

- **Code with Spec**：复杂功能开发，先生成 Spec 文档（需求描述、设计方案、任务拆解、验收标准），确认后再执行
- **Prototype Demos**：快速原型验证，从创意到可运行原型，跳过 Spec 直接执行
- **Create Tools**：构建自动化工具，描述需求即可生成可运行工具

Quest 支持本地 + Worktree + 远程并行执行，长时间运行自动恢复中断。

## 其他核心能力

- **NEXT 智能代码预测**：基于上下文预测下一步编辑，按 `⌥P` 触发
- **上下文引擎**：结合代码仓库分析与自适应记忆，支持约 10 万文件规模代码库
- **Repo Wiki**：自动生成代码库架构文档，减少 AI「幻觉」
- **Browser Agent / Code Review Agent**：专项智能体
- **MCP 协议 + Skills + Hooks**：可扩展工具链
- **多形态**：IDE + CLI + JetBrains 插件
