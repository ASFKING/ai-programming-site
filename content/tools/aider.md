> 开源终端 AI 编程工具，与 Claude Code 直接竞争

---

## 核心功能

### Git 原生集成

- 直接在 Git 仓库中工作
- 自动提交代码变更
- 支持代码审查模式

### 多模型支持

- 支持 Claude、GPT-4o、Gemini、本地模型等
- 可随时切换模型
- 推荐使用 Claude 3.5 Sonnet

### 终端原生

- 完全在终端中运行
- 与现有工作流无缝集成
- 轻量不占资源

---

## 适用场景

- ✅ 终端 AI 编程（Claude Code 的开源替代）
- ✅ Git 工作流深度集成
- ✅ 需要自定义模型的开发者
- ✅ 预算有限但想用 Agent 编程

## 局限性

- 需要自备 LLM API Key
- 配置有一定门槛
- 没有 GUI
- 依赖终端环境
